# Climapp

Oi! Seguem as instruções de uso. Este protótipo foi feito em Vue 2, 
portanto também pode ser gerenciado via UI (:

## Setup do Projeto:
```
npm install
```

### Gerenciamento por GUI
```
vue ui
```

### Modo desenvolvimento
```
npm run serve
```

### Modo produção
```
npm run build
```

### Lints e fixes
```
npm run lint
```
